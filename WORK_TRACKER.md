# WORK TRACKER - 记录工作结果

此文件用于跟踪任务的执行结果，包括成功或失败的原因。

## 工作记录格式

每项工作应按以下格式记录：

```
### [日期时间] - [任务描述]
- 状态: [进行中/成功/失败]
- 执行时间: [开始时间 -> 结束时间]
- 结果详情:
  - 成功: [成功完成的具体内容]
  - 失败: [失败原因分析]
- 影响: [对后续工作的潜在影响]
```
